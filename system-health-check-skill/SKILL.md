# 系统健康检查技能

## 功能概述
本技能提供全面的Linux系统健康检查能力，包括CPU、内存、磁盘、网络和系统日志分析。

## 核心能力
1. **CPU状态监控** - 使用率、负载均衡、进程分析
2. **内存分析** - 内存使用、交换分区、缓存状态
3. **磁盘检查** - 使用率、IO性能、inode使用情况
4. **网络诊断** - 连接状态、端口监听、带宽使用
5. **系统日志** - 错误日志、安全事件、系统消息

## 使用流程
1. 收集系统基础信息
2. 执行各项健康检查
3. 分析检查结果
4. 生成诊断报告
5. 提供优化建议

## 依赖要求
- Python 3.6+
- psutil 库
- 必要的系统权限（部分检查需要sudo）

## 输出格式
- JSON格式的详细数据
- 人类可读的彩色报告
- 告警级别的标识（INFO/WARNING/CRITICAL）
